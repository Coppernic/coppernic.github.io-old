Libraries#

Coppernic SDK is a collection of Android libraries to help developer in using peripherals of Coppernic's products. These libraries are hosted on an maven instance available at nexus.coppernic.fr. You can browse in libs-release-coppernic repository and check out what are the last version of our libs.
